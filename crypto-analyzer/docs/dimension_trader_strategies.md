## 二、核心指标说明

这三个指标贯穿所有策略：

**gradient(cs, n)** — n 根 K线的平均价格变化率（近似斜率）  
正值 = 上涨趋势，负值 = 下跌趋势

**flux(cs, n)** — 近 n 根 K线的买方成交量占比（Taker Buy Volume / Total Volume）  
> 0.5 = 买方主导，< 0.5 = 卖方主导，通常用 0.47~0.53 做阈值

**amplitude(cs, n)** — 近 n 根 K线的平均振幅（(high-low)/close）  
衡量市场波动烈度，越高越不稳定

## 五、DecelBounce 家族（E16-E30）—— 全部 LONG

**手工设计 + 86 个标的 70/30 走时验证，测试胜率 57%~69%**  
**标的：全量（BIG4 + ALT99，约 90 个）**

### 核心逻辑（所有 E16-E30 共享）

**DecelBounce** = "减速反弹"。在 4h 宏观上升趋势中，捕捉 1h 的短暂回调结束、重新翻正的入场点。

```
4h gradient(mac_n) > 0.003   →  4h 宏观上升趋势（4h 在涨）
1h gradient(h_n)  < -hist_th →  1h 已连续回调 h_n 根（已经跌够了）
1h gradient(2)    > 0         →  1h 最近 2 根刚刚翻正（底部反转确认）
flux(2)           > f_min     →  买压回升（默认 > 0.53，资金回流）
```

可选过滤器：
- `amp1_max`：振幅不能太大（排除暴跌行情，只要有序回调）
- `amp4_min`：4h 需要一定振幅（确保宏观趋势有力度）
- `hist_th`：下行幅度要求（值越大，要求回调越陡）

各策略之间的差异就是 **h_n（回调深度）和过滤条件** 不同，以适应不同节奏的市场。
