2.3 全局限制

不设止盈/止损/持仓时间

开启后 live / whale / bigmid 三个策略 新开仓不设 SL / TP / 超时平仓，进程内硬止盈与移动止盈检查也会跳过。
仅影响新开仓，存量仓位保持原 SL/TP/超时。修改后需重启策略进程生效。

跳过 5m 阴/阳收盘确认（限价单）

开启后 live / whale / bigmid / f3 限价单触发即按 limit_price 成交，不再等下根 5m K 线收盘验阴/阳，不再产生 TRIGGER_OBSERVING / 5M_REJECT 事件。
遇上反弹中追空可能立刻被反向打到 SL；建议 paper 验证 2-3 天再判断。修改后需重启策略进程生效。

CHASE 允许慢爬入场

开启后 strategy_live 的 chase-entry 不再要求"窗口内至少一根 5m bar 涨幅 ≥ 3%"，慢爬币也允许追多（其它守卫如 24h 区间、bar_age、entry_position 仍生效）。
注意：chase 的 TP/SL 针对急涨快回设计，慢爬币更易 timeout/亏小单。修改后需重启策略进程生效。

入场总开关 (chase / dump)

近 7 天数据: chase-entry 高位追多 -789U / dump-entry 低位杀跌 -288U，"顺势追突破"在加密假突破频发市场结构性反向。关掉后已有持仓继续 monitor SL/TP, IDLE 不再触发新入场。60s 内动态生效，不需要重启。
